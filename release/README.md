#### 使用说明

1.	安装python3.8或更新版本，安装mysql8.0或更新版本。
2.	安装好reqirements.txt里的所有库并配置selenium库的webdriver至默认路径。
3.	将所有python文件放在同一文件夹下。
4.	运行main.py文件。
5.	选择是否更新数据，选择“是”后，等待数据更新完毕，弹出“数据更新完毕”之后，关闭此弹窗，出现用户界面，或者选择否，直接进入用户界面。
